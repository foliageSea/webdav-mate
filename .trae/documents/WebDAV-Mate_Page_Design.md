# WebDAV-Mate 页面设计文档（Desktop-first）

## 0. Global Styles（全局规范）

### Layout

- 采用“顶部标题栏 + 左侧连接/路径辅助区（可折叠）+ 右侧主内容区”的桌面应用经典三栏/两栏混合布局。
- 主内容区以 CSS Grid 构建文件网格；工具栏与列表项使用 Flexbox 对齐。
- 断点（可选）：
  - ≥ 1200px：三栏完整展示（左侧连接/收藏区 + 主区 + 右侧详情/预览抽屉可选）
  - 800–1199px：左侧区域折叠为图标栏；预览改为弹窗
  - ≤ 799px：仍可用但非优先（桌面优先），保持基础可操作性

### Design Tokens

- Background: #0B1220（深色主背景） / Surface: #0F1A2B（卡片/面板）
- Primary: #3B82F6（操作主色）
- Accent (Success/Warning/Error): #22C55E / #F59E0B / #EF4444
- Border: rgba(148,163,184,0.16)
- Text:
  - Primary: rgba(255,255,255,0.92)
  - Secondary: rgba(255,255,255,0.68)
- Typography scale（建议）：12 / 14 / 16 / 20 / 24
- Radius: 10（卡片）/ 8（按钮）
- Shadow: 轻量阴影用于悬浮卡片与弹窗

### Components States

- Button：默认/hover（亮度+8%）/active（亮度-6%）/disabled（opacity 0.45）
- 卡片：hover 时边框高亮 + 轻微上浮（translateY -1）
- 拖拽：
  - 可投放目标：蓝色描边 + 背景浅高亮
  - 禁止投放：红色描边 + 禁止图标
- 选择态：卡片左上角勾选框 + 边框高亮；支持多选

### Meta Information（适用于应用窗口级别）

- Title：WebDAV-Mate
- Description：多 WebDAV 服务文件管理与批量传输工具
- Open Graph：桌面应用不强依赖；如需导出分享页，可复用上述 title/description

## 1) 连接管理页（/servers）

### Page Structure

- 上方：标题区（“连接管理”）+ 主要操作按钮（新增连接）。
- 中间：连接列表（表格或列表卡片）。
- 右侧（可选）：连接详情编辑面板（选中某连接后出现）。

### Sections & Components

1. 顶部操作栏
   - 左：页面标题、搜索框（可选，若连接较多）
   - 右：按钮组
     - “新增连接”（Primary）

2. 连接列表（列表卡片）
   - 每项包含：连接名称、URL、用户名、状态指示（最近测试结果/未知）。
   - 行内操作：
     - “设为当前”/“当前”标签
     - “测试”按钮（触发连接测试并显示结果 Toast）
     - “编辑”
     - “删除”（二次确认）

3. 新增/编辑连接弹窗（Modal）
   - 表单字段：名称、服务器 URL、用户名、密码（密码框）、基础路径（可选）。
   - 校验：URL 必填；用户名/密码必填。
   - 底部按钮：取消 / 保存 / 测试连接（可并排）。

### Interaction Notes

- 保存后自动出现在列表；可提示“已保存”。
- 测试连接时按钮进入 loading；结果用 Toast + 列表状态标记。

## 2) 文件管理页（主界面）（/files/:serverId）

### Page Structure

- 顶部：全局工具栏（连接切换、路径、刷新、上传/下载、进入传输任务）。
- 左侧：连接快捷区/目录书签区（可折叠）。
- 主区：文件网格（Grid）。
- 覆盖层：拖拽上传提示层（Drag Overlay）。
- 预览：图片/视频预览弹窗或右侧抽屉（根据窗口宽度）。

### Sections & Components

1. 顶部工具栏（Sticky）
   - 左：返回上级、刷新
   - 中：当前连接下拉（快速切换） + 面包屑路径（可点击跳转）
   - 右：按钮组
     - “上传”（选择文件/文件夹）
     - “下载”（对已选项可用）
     - “传输任务”（进入 /transfers，并显示进行中数量 badge）
     - “选择模式”切换（可选：勾选多选/普通点击）

2. 文件网格（核心）
   - 布局：CSS Grid，自适应列宽（如 minmax(160px, 1fr)）。
   - 每个文件卡片：
     - 顶部：缩略图/图标（文件夹/图片/视频/通用）
     - 中部：文件名（两行截断）
     - 底部：大小、修改时间（次要信息）
     - 左上：选择框（显示于 hover/选择模式）
   - 交互：
     - 单击：选择（或打开预览/进入文件夹）
     - 双击：进入文件夹；对媒体文件则打开预览
     - 右键：上下文菜单（下载/删除/重命名/移动）

3. 拖拽能力
   - 本机 → 网格：显示 Overlay（“松开以上传到当前目录”）；释放后创建上传任务。
   - 网格内 → 文件夹卡片：目标卡片高亮；释放后触发“移动到该文件夹”。
   - 拖拽冲突：同名提示（交由传输任务页或弹窗策略）。

4. 批量操作条（Selection Bar）
   - 在多选时出现，显示：已选数量、下载、删除、取消选择。

5. 图片/视频预览
   - 弹窗（Modal）或右侧抽屉（Drawer）：
     - 标题：文件名
     - 内容：图片（自适应缩放）/ 视频（<video> 控件）
     - 底部：上一张/下一张（仅媒体文件）+ “在文件夹中定位”（可选）

### Responsive Behavior

- 宽屏：预览用右侧抽屉，保持网格可见。
- 中屏：预览用弹窗，避免压缩网格。

## 3) 传输任务页（/transfers）

### Page Structure

- 顶部：标题区 + 全局操作（暂停全部/继续全部可选、清理完成）。
- 主区：任务列表（按上传/下载分组或按时间排序）。
- 底部（可选）：总体进度汇总条。

### Sections & Components

1. 顶部操作栏
   - 左：标题“传输任务” + 返回文件管理页
   - 右：按钮
     - “清理已完成”（次要按钮）

2. 任务列表（List）
   - 每条任务展示：
     - 方向图标（上传/下载）+ 文件名
     - 远端路径/本地路径（次要信息，单行截断）
     - 进度条（百分比 + transferred/total）
     - 状态标签（排队/进行中/暂停/完成/失败）
   - 行内控制：暂停/继续、取消、重试（失败时显示）。

3. 错误展示
   - 失败任务展开后显示错误原因文本；提供“复制错误信息”按钮（便于反馈）。

### Interaction Notes

- 任务完成后保留记录，便于回溯；清理操作仅清理完成/取消项。
- 失败重试将以同参数重新入队；如果同名冲突需提示覆盖/跳过/重命名。
